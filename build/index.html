bliss

Blog it simple, stupid!

bliss is a moderately minimal blog generator, with a build script below 100 lines of shell. It's forked from Bradley Taunt's barf, and keeps things mostly the same while adding in a few little creature comforts for my own use.

You can learn more by reading the README.
